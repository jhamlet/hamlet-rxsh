hamletink/rxsh
=================

> Observable versions of common shell commands

Installation
------------

```
npm install --save @hamletink/rxsh
```

Usage
-----

```js
import { sh } from '@hamletink/rxsh';

// or

import { sh } from '@hamletink/sh';
```
